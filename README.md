# AI Study Assistant

An AI-powered study assistant built with Python and Streamlit.

## Features
- Upload PDF notes
- Extract text from PDFs
- Generate AI-powered summaries using Google Gemini
- Clean and user-friendly interface

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- PyPDF2
- python-dotenv

## Future Features
- AI Quiz Generator
- Flashcards
- Translation
- Ask Questions about Notes
- Text-to-Speech
