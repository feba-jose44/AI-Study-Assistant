# 🤖 AI Study Assistant

An AI-powered study assistant built using Python and Azure AI.

## 🚀 Features

- 📄 Upload study notes
- 📝 Generate summaries
- ❓ Create quizzes
- 💡 Explain difficult concepts
- 🔊 Text-to-speech (coming soon)

## 🛠️ Tech Stack

- Python
- Streamlit
- Azure AI
- Git/GitHub
